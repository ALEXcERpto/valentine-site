# Valentine's Day Special Site

A romantic Valentine's Day website where you can ask someone special to be your Valentine!

## Features

- Beautiful romantic gradient design with pink and red colors
- Floating heart animations in the background
- Two-page flow:
  1. Enter your Valentine's name
  2. Ask "Can you be my Valentine?" with Yes/No options
- Special responses:
  - **Yes**: Beautiful flower emoji with romantic message and celebration hearts
  - **No**: Sad emoji with gentle message
- Fully responsive design (mobile and desktop)
- Smooth animations throughout

## How to Use

1. Open `index.html` in your browser
2. Enter the name of your Valentine
3. Click "Next"
4. The person clicks "Yes" or "No"
5. See the romantic (or sad) response!
